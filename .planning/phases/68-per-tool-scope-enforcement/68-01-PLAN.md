<objective>
Establish the per-tool scope single-source-of-truth and the import-time drift guard
(OAUTH-06 SC#4). Add a `TOOL_SCOPE_REQUIREMENTS: dict[str, str]` mapping co-located with
`TOOL_REGISTRY` in `tools/__init__.py`, covering ALL 16 registered tools, and a fail-loud
import-time `RuntimeError` if any registry key lacks a scope. Add a `require_scope()` helper
plus an `InsufficientScopeError` in a new `oauth/scopes.py` that Plan 02 consumes at the
dispatch layer.

Purpose: A single map + one import-time guard makes it structurally impossible to ship a
tool without a scope assignment (design Risk 4 closed at startup, not test-time).
Output: `oauth/scopes.py` (new), `TOOL_SCOPE_REQUIREMENTS` + guard in `tools/__init__.py`,
unit tests proving completeness, valid values, and that the guard raises.
