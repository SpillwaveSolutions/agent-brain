    - get_xdg_state_dir() returns Path.home() / ".local" / "state" / "agent-brain" when XDG_STATE_HOME is not set
    - get_xdg_state_dir() returns Path(env_val) / "agent-brain" when XDG_STATE_HOME is set
    - get_registry_path() returns XDG state dir / "registry.json" when it exists there
    - get_registry_path() falls back to legacy path if registry.json exists only there
    - get_registry_path() returns XDG path as default write target when neither exists
    - migrate_legacy_paths() copies config.yaml to XDG config dir, registry.json to XDG state dir, then deletes legacy dir
    - migrate_legacy_paths() returns False and does nothing when no legacy dir exists
    - migrate_legacy_paths() returns False when XDG dirs already exist (no double-migrate)
    - migrate_legacy_paths() catches PermissionError/OSError, warns to stderr, returns False
    - migrate_legacy_paths() prints notice to stderr on success (using click.echo(err=True))
    - migrate_legacy_paths(silent=True) suppresses notice
  </behavior>
  <action>
    Create `agent-brain-cli/agent_brain_cli/xdg_paths.py` following the research pattern exactly:

    1. Module-level constant: `LEGACY_DIR = Path.home() / ".agent-brain"`

    2. `get_xdg_config_dir() -> Path`: reads `os.environ.get("XDG_CONFIG_HOME")`, returns `Path(val) / "agent-brain"` if set, else `Path.home() / ".config" / "agent-brain"`.

    3. `get_xdg_state_dir() -> Path`: reads `os.environ.get("XDG_STATE_HOME")`, returns `Path(val) / "agent-brain"` if set, else `Path.home() / ".local" / "state" / "agent-brain"`.

    4. `get_registry_path() -> Path`: checks XDG state dir first, then legacy dir, returns whichever has `registry.json`. Default to XDG path if neither exists.

    5. `migrate_legacy_paths(*, silent: bool = False) -> bool`:
       - Return False if LEGACY_DIR doesn't exist
       - Return False if either XDG config or state dir already exists
       - Try: mkdir parents for both XDG dirs, shutil.copy2 config files to config dir, shutil.copy2 state files to state dir, shutil.rmtree legacy dir
       - On success: click.echo notice to stderr (unless silent), return True
       - On PermissionError/OSError: log warning, click.echo warning to stderr (unless silent), return False
