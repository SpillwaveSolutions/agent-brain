Create `manifest_tracker.py` with these classes following the research patterns exactly:

1. **FileRecord** dataclass: `checksum: str`, `mtime: float`, `chunk_ids: list[str]`
2. **FolderManifest** dataclass: `folder_path: str`, `files: dict[str, FileRecord]` (key = absolute file path)
3. **EvictionSummary** dataclass: `files_added: list[str]`, `files_changed: list[str]`, `files_deleted: list[str]`, `files_unchanged: list[str]`, `chunks_evicted: int`, `chunks_to_create: int`
4. **ManifestTracker** class:
   - `__init__(self, manifests_dir: Path)` — stores dir, creates `asyncio.Lock`
   - `_manifest_path(folder_path: str) -> Path` — `sha256(folder_path.encode()).hexdigest() + ".json"`
   - `async load(folder_path: str) -> FolderManifest | None` — returns None if file missing, uses `asyncio.to_thread` for IO
   - `async save(manifest: FolderManifest) -> None` — atomic write via temp + `Path.replace()`, under lock
   - `async delete(folder_path: str) -> None` — remove manifest file, under lock
   - `_read_manifest(path, folder_path) -> FolderManifest` — JSON deserialization
   - `_write_manifest(manifest) -> None` — `mkdir(parents=True, exist_ok=True)`, JSON serialization, temp+replace
5. **compute_file_checksum(file_path: str) -> str** — SHA-256 in 64KB chunks

Create `chunk_eviction_service.py`:

1. **ChunkEvictionService** class:
   - `__init__(manifest_tracker: ManifestTracker, storage_backend: StorageBackendProtocol)`
   - `async compute_diff_and_evict(folder_path: str, current_files: list[str], force: bool = False) -> tuple[EvictionSummary, list[str]]`
     - **Force mode:** Load prior manifest, collect ALL prior chunk_ids, call `delete_by_ids`, delete manifest, return all current_files for indexing
     - **No manifest:** Return all files as added, 0 chunks evicted
     - **Normal diff:** Compare current_files against prior manifest:
       - File not in prior = added (index it)
       - File in prior, mtime unchanged = unchanged (skip checksum, skip indexing)
       - File in prior, mtime changed, checksum same = unchanged (skip indexing)
       - File in prior, mtime changed, checksum different = changed (evict old chunks, index it)
       - File in prior but not on disk = deleted (evict old chunks)
     - Use `asyncio.to_thread(compute_file_checksum, fp)` for checksums
     - Bulk evict: collect all chunk_ids from deleted+changed files, call `self._storage.delete_by_ids(ids)` once
     - Return `(EvictionSummary, files_to_index)` where files_to_index = added + changed

Update `services/__init__.py` to export `ManifestTracker`, `FolderManifest`, `EvictionSummary`, `ChunkEvictionService`.

Use TYPE_CHECKING import for StorageBackendProtocol to avoid circular imports (same pattern as ContentInjector).
