Create `agent-brain-cli/agent_brain_cli/config_migrate.py` with:

1. A `MigrationResult` dataclass:
   ```python
   @dataclass
   class MigrationResult:
       original: dict[str, Any]
       migrated: dict[str, Any]
       changes: list[str]  # Human-readable change descriptions
       already_current: bool
   ```

2. A `MIGRATIONS` list of migration functions, applied in order:
   ```python
   MigrationFn = Callable[[dict[str, Any]], tuple[dict[str, Any], list[str]]]

   def _migrate_use_llm_extraction(config: dict[str, Any]) -> tuple[dict[str, Any], list[str]]:
       """Migrate graphrag.use_llm_extraction -> graphrag.doc_extractor."""
       changes: list[str] = []
       config = copy.deepcopy(config)
       graphrag = config.get("graphrag", {})
       if "use_llm_extraction" in graphrag:
           old_val = graphrag.pop("use_llm_extraction")
           if old_val:
               graphrag["doc_extractor"] = "langextract"
               changes.append(
                   "graphrag.use_llm_extraction=True -> graphrag.doc_extractor=langextract"
               )
           else:
               changes.append(
                   "graphrag.use_llm_extraction=False -> removed (default: no doc extraction)"
               )
           config["graphrag"] = graphrag
       return config, changes

   MIGRATIONS: list[MigrationFn] = [
       _migrate_use_llm_extraction,
   ]
   ```

3. `migrate_config(config: dict[str, Any]) -> MigrationResult`:
   - Deep-copy original
   - Apply each migration in MIGRATIONS sequentially
   - Collect all changes
   - Set already_current = (len(changes) == 0)
   - Return MigrationResult

4. `migrate_config_file(path: Path) -> MigrationResult`:
   - Read YAML from path
   - Call migrate_config
   - If changes, write migrated dict back to path using yaml.safe_dump with `default_flow_style=False, sort_keys=False`
   - Return result

5. `diff_config(config: dict[str, Any]) -> str`:
   - Call migrate_config to get original and migrated
   - If no changes, return ""
   - Generate unified diff between `yaml.safe_dump(original)` and `yaml.safe_dump(migrated)` using `difflib.unified_diff`
   - Return the diff string (lines joined with newline)

6. `diff_config_file(path: Path) -> str`:
   - Read YAML from path
   - Call diff_config on the parsed dict
   - Return diff string

**Part A: Add `migrate` subcommand** to config_group in config.py:

```python
@config_group.command("migrate")
@click.option("--file", "config_file", type=click.Path(exists=True), default=None,
              help="Path to config file (default: auto-detect)")
@click.option("--dry-run", is_flag=True, help="Show what would change without modifying")
def migrate_config_cmd(config_file: str | None, dry_run: bool) -> None:
    """Migrate config.yaml to the current schema version.

    Upgrades deprecated keys and restructures config sections.
    Use --dry-run to preview changes without modifying the file.

    \b
    Examples:
      agent-brain config migrate
      agent-brain config migrate --dry-run
      agent-brain config migrate --file ./old-config.yaml
    """
```

Implementation:
1. Resolve config path (--file or _find_config_file)
2. If no config found, print "[yellow]No config file found.[/]" and exit 0
3. If `--dry-run`: call `diff_config_file(path)`, print diff or "Config is already up to date", exit 0
4. If not dry-run: call `migrate_config_file(path)`
   - If already_current: print "[green]Config is already up to date[/]" and exit 0
   - If changes: print each change, then print "[green]Config migrated successfully[/] ({path})" and exit 0
