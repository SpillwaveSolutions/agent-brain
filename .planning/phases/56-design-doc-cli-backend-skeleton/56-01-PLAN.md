   - **`reset()` has no MCP tool equivalent** — v2 did not ship `reset_index` as an MCP tool (destructive op + cache invalidation makes it dangerous via MCP). Mitigation: skeleton `raise NotImplementedError`; Phase 57+ decides whether to add the tool or hold for v4.
