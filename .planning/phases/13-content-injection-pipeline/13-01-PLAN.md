**1. Create `content_injector.py`** in `agent_brain_server/services/`:

Class `ContentInjector` following the FolderManager pattern (see `folder_manager.py`):

- `__init__(self, script_path: Path | None = None, folder_metadata: dict[str, Any] | None = None)` — stores params, calls `_load_script()` if script_path provided.
- `_load_script(self, script_path: Path) -> None` — uses `importlib.util.spec_from_file_location("_injector_script", script_path)` to load the module. Validates: file exists (`FileNotFoundError`), spec loads (`ImportError`), module has `process_chunk` attribute (`AttributeError`), attribute is callable (`TypeError`). Stores `self._process_chunk_fn`.
- `from_folder_metadata_file(cls, metadata_path: Path) -> ContentInjector` — classmethod, loads JSON file, validates root is dict, returns `ContentInjector(folder_metadata=data)`.
- `build(cls, script_path: str | None = None, metadata_path: str | None = None) -> ContentInjector | None` — classmethod factory. If both None, returns None. Creates injector from whichever paths are provided (can be both). Returns ContentInjector with both script and folder_metadata set.
- `apply(self, chunk: dict[str, Any]) -> dict[str, Any]` — merges `self._folder_metadata` first (shallow merge via `{**chunk, **self._folder_metadata}`), then calls `self._process_chunk_fn(chunk)` if loaded. Both steps wrapped in try/except Exception with `logger.warning` (INJECT-05). If `process_chunk` returns non-dict, log warning and skip. After enrichment, call `_validate_metadata_values(chunk)`.
- `_validate_metadata_values(self, chunk: dict[str, Any]) -> dict[str, Any]` — iterate keys, for any value that is not `str | int | float | bool | None`, log warning and remove the key (ChromaDB requires flat scalars). Return cleaned dict.
- `apply_to_chunks(self, chunks: list, known_keys: set[str]) -> int` — convenience method that iterates chunks (TextChunk | CodeChunk), calls `apply(chunk.metadata.to_dict())`, writes new/changed keys (not in `known_keys`) back to `chunk.metadata.extra`. Returns count of enriched chunks.

Use `from __future__ import annotations`. Use `logging.getLogger(__name__)`. Use `typing.Any`, `pathlib.Path`.
