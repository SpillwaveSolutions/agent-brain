<objective>
Add retry resilience to Ollama embedding provider and fix sentence-transformers reranker startup timeout.

Separately, `agent-brain start` with sentence-transformers reranker times out on first model download because the default 30s timeout is insufficient.

Output: Modified OllamaEmbeddingProvider with retry logic, comprehensive unit tests, and increased CLI startup timeout.

    3. Add private helper method `_is_retryable_error`:
       ```python
       def _is_retryable_error(self, exc: Exception) -> bool:
           if isinstance(exc, (BrokenPipeError, ConnectionResetError)):
               return True
           if isinstance(exc, httpx.ReadTimeout):
               return True
           if isinstance(exc, httpx.RemoteProtocolError):
               return True
           if isinstance(exc, httpx.ConnectError):
               return "refused" not in str(exc).lower()
           return False
       ```

    4. Rewrite `_embed_batch` with retry loop:
       ```python
       async def _embed_batch(self, texts: list[str]) -> list[list[float]]:
           last_exc: Exception | None = None
           for attempt in range(self._max_retries + 1):
               try:
                   response = await self._client.embeddings.create(
                       model=self._model, input=texts,
                   )
                   result = [item.embedding for item in response.data]
                   if self._request_delay_ms > 0:
                       await asyncio.sleep(self._request_delay_ms / 1000)
                   return result
               except Exception as e:
                   # Non-retryable: connection refused
                   if isinstance(e, ConnectionRefusedError):
                       raise OllamaConnectionError(self._base_url, cause=e) from e
                   if isinstance(e, httpx.ConnectError) and "refused" in str(e).lower():
                       raise OllamaConnectionError(self._base_url, cause=e) from e
                   # Non-retryable: model not found
                   if "model not found" in str(e).lower():
                       raise ProviderError(f"Failed to generate batch embeddings: {e}", self.provider_name, cause=e) from e
                   # Non-retryable: unknown error type
                   if not self._is_retryable_error(e):
                       raise ProviderError(f"Failed to generate batch embeddings: {e}", self.provider_name, cause=e) from e
                   # Retryable: backoff and retry
                   last_exc = e
                   if attempt < self._max_retries:
                       delay = min(2 ** attempt, 30)
                       logger.warning(f"Retryable error in _embed_batch (attempt {attempt + 1}/{self._max_retries}): {e}. Retrying in {delay}s...")
                       await asyncio.sleep(delay)
           # All retries exhausted
           raise ProviderError(f"Failed to generate batch embeddings after {self._max_retries} retries: {last_exc}", self.provider_name, cause=last_exc) from last_exc
       ```
