## Content Gaps Found (Integration Checker)

### Gap 1 — CFGDOC-01: config.json example documents nonexistent fields

**File:** `docs/CONFIGURATION.md` (line 523 section)
**Problem:** Documented fields (`default_mode`, `default_top_k`, `default_threshold`, `include_code`, `languages`) do not exist in the actual config.json written by `agent-brain init`. Actual fields: `bind_host`, `port_range_start`, `port_range_end`, `auto_port`, `chunk_size`, `chunk_overlap`, `exclude_patterns`.
**Also:** Stale `.claude/agent-brain/` paths at lines 34, 290, 300, 525, 644.

### Gap 2 — GUIDE-02: stale path in QUICK_START.md

**File:** `docs/QUICK_START.md` line 57
**Problem:** `"This creates a .claude/agent-brain/ directory"` → should be `.agent-brain/`
**Source:** `agent-brain-cli/agent_brain_cli/commands/init.py` line 37: `STATE_DIR_NAME = ".agent-brain"`

### Gap 3 — GUIDE-03: stale paths in PLUGIN_GUIDE.md

**File:** `docs/PLUGIN_GUIDE.md`
- Line 364 (prose): `"Creates .claude/agent-brain/ with project configuration."` → `.agent-brain/`
- Line 639 (code block): `Path(".claude/agent-brain/runtime.json")` → `.agent-brain/runtime.json`
**Note:** Code block path excluded from Phase 33 link checker by design — silently survived audit.

### Gap 4 — GUIDE-05: GRAPHRAG_GUIDE.md incorrectly blocks multi+PostgreSQL

**File:** `docs/GRAPHRAG_GUIDE.md` line 143
**Problem:** States "`graph` and `multi` query modes require GraphRAG enabled with the ChromaDB backend" — this is **incorrect**. `query_service.py` lines 616–631 shows `multi` mode gracefully skips the graph component on non-ChromaDB backends without error.
**Conflict:** `agent-brain-plugin/commands/agent-brain-multi.md` (Phase 32 output) correctly documents graceful adaptation — the two docs contradict each other.

### Gap 5 — XREF-01: check_doc_links.py anchor verification bug

**File:** `scripts/check_doc_links.py` lines 97–99
**Problem:** `is_url()` includes `"#"` in `startswith()` tuple → same-file anchor links classified as URLs and skipped. **127 anchor links across 71 files were never checked.** 5 broken ToC anchors in `docs/DEVELOPERS_GUIDE.md` undetected.

### Gap 6 — XREF-03: SKILL.md root files outside Phase 33 scope

**Files:** `agent-brain-plugin/skills/using-agent-brain/SKILL.md` (421 lines), `agent-brain-plugin/skills/configuring-agent-brain/SKILL.md` (691 lines)
**Problem:** `DEFAULT_GLOBS` covers `skills/*/references/*.md` but not `skills/*/SKILL.md`. Primary skill entry points have no `last_validated` frontmatter and were not link-verified.
