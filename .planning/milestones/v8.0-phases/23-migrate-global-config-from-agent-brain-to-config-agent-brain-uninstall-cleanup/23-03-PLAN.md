  <behavior>
    - `agent-brain uninstall` with no flags prompts "Remove all Agent Brain global data? [y/N]"
    - Answering N/no aborts with "Aborted." message
    - Answering Y/yes proceeds with cleanup
    - `agent-brain uninstall --yes` skips confirmation and proceeds directly
    - Before removing directories, reads registry.json and sends SIGTERM to all running server PIDs
    - Removes ~/.config/agent-brain/ (XDG config dir)
    - Removes ~/.local/state/agent-brain/ (XDG state dir)
    - Removes ~/.agent-brain/ (legacy dir, if present)
    - Does NOT remove any .claude/agent-brain/ project directories
    - Handles missing directories gracefully (no error if dir doesn't exist)
    - Reports what was removed in summary output
    - --json flag outputs JSON result
  </behavior>
  <action>
    1. Create `agent-brain-cli/agent_brain_cli/commands/uninstall.py`:

       ```python
       @click.command("uninstall")
       @click.option("--yes", "-y", is_flag=True, help="Skip confirmation prompt")
       @click.option("--json", "json_output", is_flag=True, help="Output as JSON")
       def uninstall_command(yes: bool, json_output: bool) -> None:
       ```

       Implementation steps inside the command:
       a. Collect directories to remove: `get_xdg_config_dir()`, `get_xdg_state_dir()`, `LEGACY_DIR`. Filter to only those that `.exists()`.
       b. If no directories exist, report "Nothing to remove" and return.
       c. If not `--yes` and not `--json`: show warning listing dirs, use `Confirm.ask("Remove all Agent Brain global data?", default=False)`. Abort if declined.
       d. Stop running servers: read registry via `get_registry_path()`. For each entry with a pid, try `os.kill(pid, signal.SIGTERM)`. Catch ProcessLookupError (already dead). Wait briefly (0.5s) for clean shutdown.
       e. Remove directories: `shutil.rmtree(dir, ignore_errors=True)` for each existing dir.
       f. Report results: list removed dirs. If `--json`, output JSON dict with `removed` list and `servers_stopped` count.
       g. Use Rich console for pretty output (consistent with rest of CLI).

    2. Register in `commands/__init__.py`: add `from .uninstall import uninstall_command` to imports and `__all__`.

    3. Register in `cli.py`: add `cli.add_command(uninstall_command, name="uninstall")`.

    4. Create `tests/test_uninstall_command.py`:
       - Use `click.testing.CliRunner` to invoke the command
       - Use `tmp_path` + `monkeypatch`/`patch` to control XDG paths
       - `test_removes_global_dirs`: create fake dirs at XDG + legacy paths, invoke with --yes, verify all removed
       - `test_stops_servers_first`: create fake registry with PID, mock os.kill, verify SIGTERM sent before rmtree
       - `test_confirmation_prompt`: invoke without --yes, provide "n" input, verify dirs NOT removed
       - `test_yes_flag_skips_prompt`: invoke with --yes, verify no prompt, dirs removed
       - `test_does_not_touch_project_dirs`: create a .claude/agent-brain/ dir in tmp_path, verify it still exists after uninstall
       - `test_nothing_to_remove`: no dirs exist, verify clean exit with "Nothing to remove" message
       - `test_json_output`: invoke with --yes --json, verify JSON output with removed list
