**Wizard Step 3: Summarization Provider**
- AskUserQuestion with options: Ollama (FREE, local, llama3.2 or mistral-small3.2), Anthropic (claude-haiku-4-5-20251001), OpenAI (gpt-4o-mini), Google Gemini (gemini-2.0-flash), Grok (grok-3-mini-fast)
- If cloud provider selected, ask for API key or api_key_env reference
- Record choice as `summarization.provider`, `summarization.model`

**Wizard Step 4: Storage Backend**
- AskUserQuestion with options: ChromaDB (default, zero-ops, best for small-medium projects), PostgreSQL + pgvector (larger datasets, requires Docker or existing database)
- Record choice as `storage.backend`
- If PostgreSQL selected, note that Step 7 (PostgreSQL setup) will handle Docker/port discovery

**Wizard Step 5: GraphRAG**
- AskUserQuestion with options: No (default), Yes - SimplePropertyGraphStore (in-memory), Yes - Kuzu (persistent graph database)
- If enabled, record `graphrag.enabled: true`, `graphrag.store_type`, `graphrag.use_code_metadata: true`
- Show note: "To use graph mode, index with `agent-brain index ./src --include-code`"

**Wizard Step 6: Default Query Mode**
- Dynamically constrain options based on GraphRAG selection:
  - If GraphRAG disabled: offer hybrid (recommended), semantic, bm25
  - If GraphRAG enabled: offer hybrid (recommended), semantic, bm25, graph, multi
- Record choice as a comment in config.yaml: `# query.default_mode: hybrid` (server does not yet have a global default_mode setting, per research open question 1)
- Note in wizard output: "Use --mode flag on queries to override"

**Wizard Step 7: Write config.yaml**
- Detect config file location (project-level `.claude/agent-brain/config.yaml` if exists, else user-level `~/.agent-brain/config.yaml`)
- If existing config found, AskUserQuestion: Update existing config or create fresh? (WIZARD-08 from research: detect and merge)
- Write comprehensive config.yaml using Python one-liner (`python3 -c "import yaml; ..."`) for safe YAML serialization
- Run `chmod 600` on the config file
- Show security warning: "Never commit this file to git"

**Wizard Step 8: Verify connectivity**
- Run `agent-brain verify` to validate provider connectivity
- If verify fails, show troubleshooting guidance and offer to re-run wizard
